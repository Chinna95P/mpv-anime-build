\# ⚡ MPV Quick Reference



\## Anime Mode

CTRL + L   → AUTO  

CTRL + ;   → ON  

CTRL + '   → OFF  



\## Anime4K (Anime Only)

L          → Toggle FAST ↔ HQ  

CTRL + 1   → Mode A  

CTRL + 2   → Mode B  

CTRL + 3   → Mode C  

CTRL + 4   → Mode AA  

CTRL + 5   → Mode BB  

CTRL + 6   → Mode CA  



\## Non-Anime

CTRL + Q   → SD Clean ↔ Texture  

Q          → Force NNEDI  

W          → NNEDI Auto  



\## Info

K          → Show current profile (2s)



\## Audio

M          → 7.1 Surround  

A          → Audio Normalization
